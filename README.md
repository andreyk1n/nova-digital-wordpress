# Nova Digital WordPress Theme

## 🇺🇦 Українська версія

Це повністю кастомізована WordPress тема, створена для сучасного лендінг сайту.  
Тема розроблена без використання конструкторів сторінок (Elementor тощо), що забезпечує високу швидкість, гнучкість і чисту структуру коду.

---

## 🚀 Основні можливості

- Повністю кастомна WordPress тема
- Побудована на SCF (Options Page + Flexible Content)
- Модульна структура секцій (hero, services, cases, process, reviews, contact, footer)
- Адаптивна верстка
- Оптимізована продуктивність
- Легка кастомізація через адмінку WordPress
- Чистий та структурований код

---

## ⚙️ Інструкція з встановлення

### 1. Встановіть WordPress
Створіть чисту інсталяцію WordPress на хостингу або локально (MAMP/XAMPP).

---

### 2. Встановіть плагін

Потрібен плагін:

**WPvivid Backup Plugin**

Встановлення:

Plugins → Add New → WPvivid Backup Plugin


---

### 3. Відновлення сайту з бекапу

У цьому репозиторії знаходиться папка:


/backup/


Вона містить повний backup сайту.

#### Кроки:

1. Перейдіть в адмінку WordPress
2. Відкрийте:

WPvivid Backup → Backup & Restore

3. Завантажте файл з папки:

/backup/

4. Натисніть **Restore**

---

### 4. Готово

Після відновлення:
- Сайт повністю налаштований
- Всі сторінки створені
- SCF поля імпортовані
- Меню та секції вже заповнені контентом

---

## 🧩 Редагування контенту

Весь контент редагується через:

- SCF Options Page
- Flexible Content секції сторінок

Можна змінювати:
- Тексти
- Зображення
- Посилання
- Соціальні мережі
- Структуру секцій

---

## 💡 Примітки

- Тема призначена для лендінгів та маркетингових сайтів
- Не використовуються page builders
- Підходить для швидкого запуску проектів і кастомізації

This is a fully custom WordPress theme built for a modern landing page website.  
It is developed without page builders (like Elementor), ensuring high performance, flexibility, and clean code architecture.

---

## 🚀 Key Features

- Fully custom WordPress theme
- Built with SCF (Options Page + Flexible Content)
- Modular section system (hero, services, cases, process, reviews, contact, footer)
- Responsive design
- Performance optimized
- Easy customization via WordPress admin
- Clean and structured codebase

---

## ⚙️ Installation Guide

### 1. Install WordPress
Set up a fresh WordPress installation on your hosting or local environment (MAMP/XAMPP).

---

### 2. Install Required Plugin

Install:

**WPvivid Backup Plugin**

Go to:

Plugins → Add New → WPvivid Backup Plugin


---

### 3. Restore from Backup

Inside this repository, you will find:


/backup/


This folder contains a full website backup.

#### Steps:

1. Open WordPress admin panel
2. Go to:

WPvivid Backup → Backup & Restore

3. Upload backup file from:

/backup/

4. Click **Restore**

---

### 4. Done

After restoration:
- Website is fully configured
- All pages are prebuilt
- SCF fields are imported
